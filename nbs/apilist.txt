# seootter Module Documentation

## seootter.article

- `class Article`
    An article file tracked for SEO optimization.


- `def insert_article(session, article)`
    Insert a new article into the database.

- `def get_article_by_id(session, article_id)`
    Get an article by its ID.

- `def get_article_by_path(session, file_path)`
    Get an article by its file path.

- `def get_articles_by_website(session, website_id)`
    Get all articles belonging to a website.

- `def delete_article(session, article_id)`
    Delete an article by ID. Raises ValueError if not found.

- `def update_article(session, article_id, focus_keyword, secondary_keywords, target_goal, url)`
    Update article fields and set last_optimized to now. Raises ValueError if not found.

- `@call_parse def seo_otter_canob(website_id, keyword, similarity_threshold)`

- `def fmt_date(dt)`

## seootter.content_mapper

- `def url_to_relpath(url, base_path, site_url)`
    Convert a page URL to a relative local path, stripping site URL and .html suffix.

- `def find_source_file(rel_path, exts)`
    Find first existing source file matching the path with any of the given extensions.

- `def url_to_file_path(url, base_path, site_url)`
    Map a website URL to its local source file path.

- `def build_limax_slug_index(base_path)`
    Build slug→filepath index using Node/limax transliteration for Arabic Astro sites.

- `def build_slug_index(base_path)`
    Build slug→filepath index by reading `slug` field from frontmatter of all source files.

- `def map_all_urls_to_files(base_path, site_url, urls, mode, normalize)`
    Map page URLs to local source file paths using the specified strategy.
